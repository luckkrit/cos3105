## การประยุกต์ใช้ bash script และคำสั่งอื่นๆ

<Card>

## ดูว่าปัจจุบันมี User คนไหน ออนไลน์อยู่บ้าง ?
## ดูว่าเครื่องนี้ เปิดมานานเท่าไหร่แล้ว , ข้อมูลเรื่อง Time/Date
## ดู command ล่าสุดที่เราใช้งาน
## ดู log การ Login SSH ล่าสุดที่เกิดขึ้น
## ดูข้อมูลว่าเครื่องนี้ เป็น Linux เวอร์ชั่นไหน อย่างไร hostname อะไร
## ดูข้อมูล สถานะ Disk , IO ต่างๆ
## ดูสถานะ RAM
## ดู process มีตัวไหนกิน Resource มาก
## ดูข้อมูล การ Mount ต่างๆ ว่าเครื่องนี้ Mount อะไรไว้บ้าง
## ดู Service อะไรบ้าง ที่ถูกโหลดมาตอน Boot
## ดู Service อะไรบ้างที่ Running อยู่ตอนนี้
## ดู log ของ Service เมื่อ 6 ชั่วโมงที่แล้ว
## ดู cron job ว่าระบบตั้งเวลารันอะไรไว้บ้าง
## ดูข้อมูล Network Port ที่เปิดอยู่ หรือเชื่อมต่ออยู่
## ดูข้อมูล สถานะ การ์ด LAN , Network Interface
## ดูสถานะ firewall / iptables
## ดู arp เครื่องที่อยู่ใน subnet เดียวกัน
## ทดสอบ ping gateway / ออก Internet แบบ Advance
## การทำ cron job
